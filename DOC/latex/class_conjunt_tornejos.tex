\doxysubsection{Descripción detallada}
Representa a un conjunt de tornejos d\textquotesingle{}un circuit. 

Conté funcions bàsiques per a afegir, borrar, iniciar, finalitzar els tornejos que conté. Els emmagatzema en un diccionari, ja que no es poden repertir noms de tornejos i serà més ràpid buscar-\/los. L\textquotesingle{}identificador del diccionari es el nom del torneig.
